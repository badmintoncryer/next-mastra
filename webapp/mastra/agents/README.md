# Mastra Agents

このディレクトリにはMastraエージェントの定義が含まれています。

## 利用可能なエージェント

### 1. Chef Agent (`chefAgent`)
実用的で経験豊富な家庭料理のシェフとして動作するエージェント。
利用可能な材料を使って料理を手助けします。

### 2. CDK Report Agent (`cdkReportAgent`)
AWS CDKのプルリクエストを分析し、レポートを生成するエージェント。

**機能:**
- aws/aws-cdkリポジトリの過去24時間にマージされたPRを取得
- 各PRの内容を要約
- 技術的な重要ポイントを抽出
- Markdown形式のレポートを生成

**使用方法:**
1. UIで「CDK Report Agent」を選択
2. チャットで「過去24時間のAWS CDK PRレポートを作成して」と入力

**環境変数:**
- `GITHUB_TOKEN` (オプション): GitHub APIのレート制限を緩和するために設定可能
  - 未設定の場合は公開APIの制限内で動作（1時間に60リクエスト）
  - 設定した場合は認証済みユーザーとして動作（1時間に5000リクエスト）

**出力:**
- チャット画面にMarkdown形式のレポートが表示されます
- レポートは`/webapp/reports/`ディレクトリに保存されます（将来的な機能拡張用）

## 新しいエージェントの追加方法

1. このディレクトリに新しいエージェントファイルを作成（例: `myAgent.ts`）
2. `@mastra/core/agent`から`Agent`クラスをインポート
3. エージェントを定義してエクスポート
4. `../index.ts`でMastraインスタンスに登録
5. 必要に応じて`/app/api/chat/route.ts`と`/app/assistant.tsx`を更新

## ツールの統合

エージェントは専用のツールを持つことができます。
ツールは`../tools/`ディレクトリで定義してください。

例: `../tools/githubTools.ts`
